# COMO CITAR HTML NO CSS - Estudo para prova

### Seletores
> Um seletor pode ser um simples marcador HTML, segue abaixo algumas:
```css
seletor {
    propriedade: valor;
}

body {
    background: yellow;
}
```

- Selecionando classes
```css
p.right {
    text-align: center;
}
```

- Selecionando id
```css
#titulo-principal {
    color: green;
}
```
